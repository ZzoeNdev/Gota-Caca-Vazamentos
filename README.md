<h1><b>Landing Page - Gota Caça Vazamentos</b></h1>
<p>Este é um projeto de landing page desenvolvido para uma empresa especializada em Caça-Vazamentos. O foco principal da aplicação é a conversão direta, facilitando o contato imediato de clientes que enfrentam problemas hidráulicos urgentes.</p>
<p>A empresa precisava de uma presença digital que servisse como um funil de vendas direto. O site foi otimizado para que o usuário encontre o serviço e tenha canais de comunicação rápidos à disposição, reduzindo a distância entre o problema e a solução.</p>
